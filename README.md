# Guessing Game (Not fully finished)

This is just a simple guessing game where I practice classes, tuple unpacking, and other python basics. 
# To play:
First choose a level, The levels are
1. Baby: 0-50 with infinite tries
2. Easy: 0-50 with 25 tries
3. Medium: 0-100 with infinite tries
4. Hard: 0-100 with 10 tries
5. Extreme: 0-200 with 10 tries
6. Demon: 0-1000 with 11 tries

Once you have chosen a level you need to pick a number in the range that is provided, if you get 3 invalid attempts in a row, the game will end. 
You can also type "quit" at any moment to stop playing and have the answer given.

There will be an option to play again at the very end, enter Y to play again or N (or anything else) to quit.
